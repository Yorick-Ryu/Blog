### Markdown All in On 插件

#### 快捷键

```
Key    Command
Ctrl + B    粗体
Ctrl + I    斜体
Alt + S    删除线
Ctrl + Shift + ]    标题(uplevel)
Ctrl + Shift + [    标题(downlevel)
Ctrl + M    Toggle math environment/进入数学环境
Alt + C    Check/Uncheck task list item
```
**实例：**  

**粗体**
*斜体*
~~删除线~~
$x+y=z$
